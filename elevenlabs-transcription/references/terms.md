## STT関連

| 用語 | 説明 |
|------|------|
| STT | Speech-to-Text。音声認識、文字起こし |
| Whisper | OpenAIのオープンソース音声認識モデル |
| Deepgram | 音声認識API。Nova-2が最新モデル |
| ElevenLabs | 音声合成で有名。文字起こしAPIもある |
| CER | Character Error Rate。文字誤り率 |
