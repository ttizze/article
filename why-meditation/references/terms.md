# 用語集

記事で使用するパーリ語・仏教用語の解説。

## 瞑想関連

| 用語 | パーリ語 | 意味 |
|------|----------|------|
| 瞑想 | bhāvanā（バーヴァナー） | 心を育てる訓練。発達・育成・修習 |
| ヴィパッサナー | vipassanā | 観察瞑想。vi（ありのままに）+ passanā（観る） |
| サマタ | samatha | 止の瞑想。心を一点に集中させる |
| サティ | sati | 気づき・念・マインドフルネス |
| ラベリング | - | 観察対象を言葉でメモする技法 |

## 三相（tilakkhaṇa）

| 用語 | パーリ語 | 意味 |
|------|----------|------|
| 無常 | anicca（アニッチャ） | すべては変化し続ける |
| 苦 | dukkha（ドゥッカ） | 変化するものに満足はない |
| 無我 | anattā（アナッター） | コントロールする「自分」はいない |

## 仏教用語

| 用語 | パーリ語 | 意味 |
|------|----------|------|
| ブッダ | Buddha | 目覚めた人 |
| 禅定 | jhāna（ジャーナ） | 深い集中状態 |
| 八正道 | ariya aṭṭhaṅgika magga | 悟りに至る8つの実践 |
| 執着 | upādāna（ウパーダーナ） | 掴もうとする心の動き |

## 流派・人物

| 用語 | 説明 |
|------|------|
| テーラワーダ仏教 | 上座部仏教。スリランカ・ミャンマー・タイなどで伝承される初期仏教 |
| マハーシ式 | マハーシ・サヤドーが体系化したヴィパッサナー瞑想法 |
| マハーシ・サヤドー | 20世紀ビルマの僧侶。近代ヴィパッサナー運動の中心人物 |
